# School 21 — учебные проекты

Проекты, выполненные в рамках обучения в Школе 21 (Школа цифровых инженеров от Сбера). Программа построена на самостоятельной разработке и парном код-ревью между студентами, без лекций.

## Бизнес-анализ (BSA)

### [BSA00_Decomposition](./BSA00_Decomposition)
Знакомство с жизненным циклом разработки ПО, типовым составом команды и функциями каждого участника на разных этапах. Практика: определение источников данных, составление глоссария, задачи на декомпозицию.

### [BSA01_Stakeholders](./BSA01_Stakeholders)
Кто такие стейкхолдеры и почему они важны для IT-систем, техники их выявления и методы работы с ними.

### [BSA02_Requirements](./BSA02_Requirements)
Требования к ПО: виды, уровни, взаимосвязи и зависимости. Модели as is и to be, построение контекстной диаграммы, определение ролей стейкхолдеров, их проблем и потребностей, выявление бизнес-требований продукта.

### [BSA03_HowToRequirements](./BSA03_HowToRequirements)
Способы выявления требований и выбор подходящего метода в зависимости от ситуации. Практика: сбор бизнес-требований, построение функциональной ролевой модели, документирование требований.

### [BSA04_Domains](./BSA04_Domains)
Понятие «домена» и выявление доменных сущностей. Построение System Information Model и её проверка с помощью CRUD-операций.

### [BSA05_Diagrams](./BSA05_Diagrams)
Моделирование требований, базовые модели и представления: диаграмма потоков данных (DFD), swimlane-диаграмма, диаграмма состояний, таблицы решений.

## C / C++

### [S21_stringplus](./S21_stringplus)
Собственная реализация библиотеки `string.h` на языке C с дополнениями, включая собственную реализацию функции `sprintf`.

### [S21_decimal](./S21_decimal)
Реализация библиотеки `s21_decimal.h` на C, добавляющей в язык поддержку типа `decimal`, отсутствующего в стандарте.

### [S21_matrix](./S21_matrix)
Библиотека для работы с числовыми матрицами на языке C.

## Go

### [S21_Go_P01](./S21_Go_P01)
Командный проект — консольная «rogue-like» игра на Go с использованием библиотеки `goncurses`, вдохновлённая классической игрой Rogue (1980).

### [S21_Go_T01](./S21_Go_T01)
Введение в язык Go: синтаксис, базовые типы данных (int, string, slice, map, struct), операции ввода-вывода. Небольшие консольные утилиты.

### [S21_Go_T02](./S21_Go_T02)
Механизмы конкурентности в Go: синхронизация и коммуникация между горутинами через channels, пакет `sync` (WaitGroup, Mutex, Once, RWMutex).

### [S21_Go_T03](./S21_Go_T03)
Веб-приложение на Go с использованием `net/http`.
