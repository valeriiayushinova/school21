# School 21 — учебные проекты

Проекты, выполненные в рамках обучения в Школе 21 (Школа цифровых инженеров от Сбера). Программа построена на самостоятельной разработке и парном код-ревью между студентами, без лекций.

## C / C++

### [S21_stringplus](./S21_stringplus)
Собственная реализация библиотеки `string.h` на языке C с дополнениями, включая собственную реализацию функции `sprintf`.

### [S21_decimal](./S21_decimal)
Реализация библиотеки `s21_decimal.h` на C, добавляющей в язык поддержку типа `decimal`, отсутствующего в стандарте.

### [S21_matrix](./S21_matrix)
Библиотека для работы с числовыми матрицами на языке C.

## Go

### [S21_Go_P01](./S21_Go_P01)
Командный проект — консольная «rogue-like» игра на Go с использованием библиотеки `goncurses`, вдохновлённая классической игрой Rogue (1980).

### [S21_Go_T01](./S21_Go_T01)
Введение в язык Go: синтаксис, базовые типы данных (int, string, slice, map, struct), операции ввода-вывода. Небольшие консольные утилиты.

### [S21_Go_T02](./S21_Go_T02)
Механизмы конкурентности в Go: синхронизация и коммуникация между горутинами через channels, пакет `sync` (WaitGroup, Mutex, Once, RWMutex).

### [S21_Go_T03](./S21_Go_T03)
Веб-приложение на Go с использованием `net/http`.
